# XKeen Android

Мобильное приложение для управления [xkeen](https://github.com/Corvus-Malus/XKeen) (Xray) на роутерах Keenetic.

Подключается к роутеру напрямую по SSH — без промежуточных серверов, без облака.

## Возможности

### Управление серверами
- Добавление прокси-серверов через VLESS-ссылки (Vision + XHTTP packet-up)
- Пакетный импорт — несколько ссылок за раз
- Удаление серверов с автотестом конфига
- Автоматическая настройка балансировки при добавлении 2+ серверов

### Маршрутизация
- **Три пресета**: стандартный (RU direct), всё через VPN, всё напрямую
- **QUIC-блокировка** — toggle для решения проблем с YouTube
- **Особые маршруты** — добавление IP/подсетей для принудительного направления через VPN или напрямую
- **Пресет Aqara** — решение проблем с IoT-устройствами Aqara при блокировках ТСПУ
- Переключение auto (leastping) / manual режимов
- Выбор серверов для балансировки

### Мониторинг
- Статус Xray, CPU, RAM с прогресс-барами
- Здоровье серверов — цветовая индикация, количество запросов
- Внешний IP роутера
- Просмотр access/error логов
- **Диагностика** — проверка xray, конфига, DNS, DNS-подмены (DPI), интернета, geosite-баз

### Инструменты
- **Мастер первоначальной настройки** — полная конфигурация xkeen после установки
- **Бекап/восстановление** конфигов на роутере
- **Копирование конфига** между роутерами
- **Обновление geosite/geoip** баз
- **Список устройств** в сети (ARP)
- Несколько профилей роутеров с псевдонимами и быстрым переключением

## Стек

- Kotlin + Jetpack Compose + Material Design 3
- JSch (SSH)
- Room (локальная БД профилей)
- Hilt (DI)

## Сборка

```bash
# Требуется JDK 17 + Android SDK
./gradlew assembleDebug
```

APK: `app/build/outputs/apk/debug/app-debug.apk`

## Как работает

Приложение не модифицирует системные настройки роутера. Оно только:
1. Читает/пишет JSON-конфиги Xray в `/opt/etc/xray/configs/`
2. Читает логи из `/opt/var/log/xray/`
3. Запускает штатные команды: `xray run -test`, `xkeen restart`, `xkeen -ug`
